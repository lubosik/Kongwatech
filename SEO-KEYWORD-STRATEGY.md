## Part 8: The Zapier Content Playbook Applied to AI Consultancy

### How Zapier Built Dominance Through Programmatic Content

Zapier built one of the largest organic content empires in SaaS through a simple insight: every combination of two tools is a search query. "How do I connect Salesforce to Mailchimp?" is a query with buyer intent. Multiply that by thousands of integration pairs and you have hundreds of thousands of pages, each capturing a small audience that collectively becomes enormous.

The same principle applies to AI consultancy in 2026, with a slightly different formula:

**Article schema (blog posts):**
Include author (with Person schema reference), datePublished, dateModified, and keywords fields. The keywords field helps AI systems understand topical relevance.

### How to Write Content That LLMs Cite

LLMs cite content that is:

1. **Unambiguously factual at the sentence level.** Write in short declarative sentences. "Claude Code is a terminal-based AI coding assistant made by Anthropic." is citable. "Claude Code is kind of like a smart helper that writes code" is not.

2. **Structured with clear headers.** AI systems parse HTML structure. H2 and H3 headings tell the model what each section is about. Use keyword-rich headings that match the questions users ask.

3. **Specific about location and context.** "In the UK, Claude Code pricing is..." or "For small businesses in Kent..." adds geographic specificity that makes the content more citable for UK-specific queries.

4. **Original in its observations.** AI systems are trained to cite sources with information not found elsewhere. If every post is a rephrasing of existing content, it will not be cited. Lubosi's personal practitioner experience, client case studies (anonymised), and documented experiments are exactly the type of original information that gets cited.

5. **Comprehensive on its topic.** A post that answers every dimension of a question (what is it, how does it work, how much does it cost, who should use it, what are the alternatives) is far more likely to be cited than a short piece that covers one angle.
